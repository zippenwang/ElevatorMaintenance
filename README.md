# ElevatorMaintenance
电梯维保管理系统Android移动端

### 项目背景
电梯维保一般分为日常维护和故障急修两种类型，为方便维修人员区分维保类型，以及让管理员更轻松实现对维保人员的管理，推出此款客户端、Web端软件。


### 项目简述
Android客户端结合Web管理端，实现对电梯维修保养（后简称为电梯维保）人员的管理。

### 所用技术
- MVP架构
- OkHttp网络请求框架
- Glide图片加载框架
- fastjson
- baidumapSDK
- Token机制实现用户认证
- 个推平台实现消息推送

- 在hongyang大神封装的OkHttpUtils基础上的二次封装，将响应码作为参数返回，便于区分异常类型
